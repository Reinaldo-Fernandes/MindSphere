# MindSphere
MindSphere é um ecossistema de procutivadade focado em neurodivergência. Une o Método Pomodoro à IA Orbit, que utiliza linguagem não-invasiva e o "Goblin Mode" para decompor tarefas. Com design baseado em cromoterapia e  mixer de som 8D, o projeto elimina a pressão tradicional, criando um espaço para florescer.
